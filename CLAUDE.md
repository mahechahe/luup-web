```bash
npm run build     # Production build

```

To add shadcn/ui components:
```bash
npx shadcn@latest add <component-name>
```

## Architecture

React 19 SPA using Vite + Tailwind CSS v4 + React Router v7.

**Path alias:** `@` maps to `./src` — use `@/...` for all internal imports.

**Auth flow:**
- `AuthContext` (`src/App/auth/AuthContext.jsx`) holds global auth state, reads from `localStorage` on mount, exposes `login()` / `logout()` / `isAuthenticated`.
- `ProtectedRoute` (`src/App/auth/ProtectedRoute.jsx`) redirects to `/login` when unauthenticated.
- Currently uses a placeholder token (`"fake-jwt-token"`) — real API integration goes in `src/App/service/authService.js`.
- `AuthProvider` must wrap the router; it suppresses rendering until the initial token check completes (`loading` state).

**Routing** (`src/App/router/AppRouter.jsx`): `/login` → Login page, `/dashboard` → protected Dashboard, `*` → Login fallback.

**UI components** live in `src/components/ui/` and are generated by shadcn/ui (new-york style, JSX not TSX, CSS variables, neutral base color, lucide icons). Do not hand-edit these files — re-run `npx shadcn@latest add` to regenerate or update them.

**Styling:** Tailwind CSS v4 is integrated via the `@tailwindcss/vite` plugin (no `tailwind.config.js`). CSS entry point is `src/index.css`. Utility merging uses `clsx` + `tailwind-merge` via `src/lib/utils.js`.
